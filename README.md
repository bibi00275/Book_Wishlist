# Book_Wishlist

**Dependencies**

Python - Programming Language
Flask - The framework used
SQLAlchemy - ORM
Pip - Dependency Management

**Install all project dependencies using:**

pip install -r requirements.txt

**Running**

FLASKAPP=main.py flask run
